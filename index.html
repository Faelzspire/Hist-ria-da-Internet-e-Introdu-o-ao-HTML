História da Internet

A internet surgiu no final da década de 1960, com o desenvolvimento da ARPANET, uma rede criada pelo Departamento de Defesa dos Estados Unidos. Ela foi projetada para ser descentralizada, garantindo comunicação mesmo em caso de ataques.

O protocolo TCP/IP, criado nos anos 1970 por Vinton Cerf e Robert Kahn, tornou possível a conexão entre redes diferentes. Em 1989, Tim Berners-Lee introduziu a World Wide Web, uma revolução que facilitou o acesso às informações.

Eventos Importantes

  1. 1969: Criação da ARPANET.
  2. 1970: Introdução do protocolo TCP/IP.
  3. 1989: Invenção da World Wide Web.
  4. 1990: O primeiro site é publicado.

Tags HTML Explicadas

As seguintes tags foram utilizadas neste projeto:

  • <h1> até <h6>: Usadas para criar títulos com hierarquia.
  • <p>: Define parágrafos de texto.
  • <mark>: Destaca trechos importantes.
  • <small>: Aplica texto menor.
  • <i>: Adiciona texto em itálico.
  • <u>: Sublinha o texto.
  • <strong>: Dá ênfase importante ao texto.
  • <ol> e <ul>: Criam listas ordenadas e não ordenadas.
  • <li>: Representa itens em listas.
  • <a>: Adiciona links.
  • <hr>: Insere uma linha horizontal.
  • <sub> e <sup>: Criam texto sobrescrito e subscrito.
  • <blockquote>: Cita trechos longos.

Novas Tags no HTML

Além das tags conhecidas, algumas novas foram exploradas:

  • <font>: Define a aparência do texto, como cor e tamanho. Exemplo: Este texto está azul e maior!
  • <del>: Marca texto excluído. Exemplo: Este texto foi removido.
  • <abbr>: Mostra abreviações. Exemplo: WWW.
